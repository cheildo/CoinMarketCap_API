# CoinMarketCap_API
An API endpoint to return the circulating supply for Scarquest SCAR token

# How to install
npm install
npm start

# API endpoints
Circulating Supply endpoint /api/scar/circulating

Total Supply endpoint api/scar/total


Deployed on Heroku
